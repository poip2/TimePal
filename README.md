# TimePal

一个智能时间管理助手

## 项目介绍

TimePal 是一个基于 AI 的时间管理工具，帮助用户更好地规划和管理时间。

## 功能特性

- 智能任务调度
- 时间追踪
- 效率分析
- 提醒通知

## 技术栈

- 前端：cd /home/poip/RooCode/TimePal/frontend && flutter run -d web-server --web-port=3001

- 后端：cd /home/poip/RooCode/TimePal/backend && npm run dev

- 数据库：posrgresql
